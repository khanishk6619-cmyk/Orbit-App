Got questions?
Everything you
need to know.
Does my data save when I close the browser? +

Yes! All your data is automatically saved to your browser's localStorage. This means your journals, workouts, sleep logs, habits, and settings persist indefinitely — even after closing the browser, restarting your computer, or going offline. Your data stays until you manually clear your browser's local storage.

Do I need an Anthropic API key? +

No! The vast majority of features — all tracking, charts, habits, sleep, workouts, journal, tasks — work perfectly without any API key. The AI features (daily check-in coaching, journal pattern analysis, monthly insights) require a free Anthropic API key. You can get one at console.anthropic.com. The app even tells you where to paste it.

Does it work on my phone? +

Absolutely. New Life is built mobile-first with a full tab bar, touch-friendly interactions, and a design that feels like a native app. For the best experience, open it in your phone's browser and tap "Add to Home Screen" — it'll behave exactly like a real app with no browser chrome.

Is it really completely free? What's the catch? +

There is no catch. New Life is 100% free and open source under the MIT License. There's no freemium tier, no subscription, no "pro" version, no in-app purchases. The only optional cost is an Anthropic API key if you want AI features — and even that has a generous free tier that covers most personal use.

Can I customize it to fit my needs? +

Yes — and that's one of the best things about it. The entire app lives in a single HTML file. Open it in any text editor (VS Code, Notepad, anything) and you can modify colors, add new features, change how things work, or remove screens you don't need. You don't need to know React deeply — most things are clearly labeled in the code.

Can I use it without an internet connection? +

Once you've downloaded the file, yes — almost everything works offline. All tracking, journaling, workouts, sleep, habits, and charts work with zero internet. The only things that need a connection are AI features (Claude API calls) and the Google Fonts on the landing page. The app itself is fully self-contained.
